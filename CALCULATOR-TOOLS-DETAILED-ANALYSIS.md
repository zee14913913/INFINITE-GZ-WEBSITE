### 1. DSR / 贷款能力计算器 (Loan Affordability Calculator)

#### 📌 **为什么最受欢迎？**

1. **直接解决核心痛点**：用户想知道"我能借多少钱？"
2. **降低决策门槛**：无需咨询顾问即可获得初步评估
3. **建立信任**：专业计算器展示公司专业性
4. **数据驱动决策**：清晰的数据帮助用户做出明智选择

#### ⚙️ **详细功能规格**

**输入项（Input）：**
```
1. 收入信息 (Income Information)
   - 月总收入 (Gross Monthly Income): RM X
   - EPF扣除 (EPF Deduction): 自动计算 8% 或手动输入
   - 所得税 (Income Tax): RM X / 月
   - SOCSO: RM X / 月
   - 其他扣除 (Other Deductions): RM X / 月
   - 净收入自动计算 (Auto-calculated Net Income)

2. 现有债务承诺 (Existing Commitments)
   - 房贷月供 (Housing Loan): RM X
   - 车贷月供 (Car Loan): RM X
   - 个人贷款月供 (Personal Loan): RM X
   - PTPTN月供 (PTPTN): RM X
   - 信用卡管理（多张）
     * 信用卡1: 已用额度 / 总额度
     * 信用卡2: 已用额度 / 总额度
     * 可动态添加/删除
     * 自动计算5%月供承诺

3. 新贷款需求 (New Loan Requirements)
   - 贷款类型 (Loan Type):
     * 个人贷款 (Personal Loan)
     * 房屋贷款 (Housing Loan)
     * 汽车贷款 (Auto Loan)
     * 营运资金贷款 (Business Loan)
   - 贷款金额 (Loan Amount): RM X
   - 利率 (Interest Rate): X% p.a.
   - 贷款期限 (Loan Tenure): X 年
```

**输出项（Output）：**
```
1. 当前财务状况 (Current Financial Status)
   - 月净收入 (Monthly Net Income): RM X
   - 现有月供总额 (Total Existing Commitments): RM X
   - 当前DSR (Current DSR): X%
   - DSR状态评级 (DSR Status):
     * Excellent (0-30%)
     * Good (31-50%)
     * Fair (51-60%)
     * Risky (61-70%)
     * Critical (71%+)

2. 新贷款影响分析 (New Loan Impact)
   - 新月供金额 (New Monthly Payment): RM X
   - 总月供金额 (Total Monthly Commitment): RM X
   - 新DSR (New DSR): X%
   - DSR增加幅度 (DSR Increase): +X%
   - 可负担性评级 (Affordability Rating)

3. 贷款详情 (Loan Details)
   - 贷款总额 (Total Loan Amount): RM X
   - 总利息 (Total Interest): RM X
   - 总还款额 (Total Payment): RM X
   - 月供金额 (Monthly Installment): RM X

4. 银行对比（可选增强功能）
   - 8家银行DSR限制对比
   - 各银行审批可能性
   - 推荐银行列表

5. 建议与推荐 (Recommendations)
   - 基于DSR状态的个性化建议
   - 优化建议（如需降低DSR）
   - 下一步行动指引
```

**计算公式：**
```
1. 净收入计算：
   Net Income = Gross Income - EPF - Income Tax - SOCSO - Other Deductions

2. 信用卡月供计算：
   Credit Card Monthly Commitment = Total Used Limit × 5%
   （银行统一假设5%，而非实际最低还款额）

3. 现有DSR计算：
   Current DSR = (Total Existing Commitments / Net Income) × 100

4. 新月供计算（减余法）：
   Monthly Rate = Annual Rate / 12
   Monthly Payment = Principal × [r(1+r)^n] / [(1+r)^n - 1]
   其中：r = 月利率, n = 总月数

5. 新DSR计算：
   New DSR = [(Existing Commitments + New Monthly Payment) / Net Income] × 100

6. 最大可贷额估算：
   Max Monthly Payment = Net Income × (DSR Limit / 100) - Existing Commitments
   反向计算最大本金（基于标准利率和期限）
```

#### 💰 **客户获利点**

**直接价值：**
1. **节省时间**：无需手动计算，30秒获得准确结果
2. **避免错误决策**：清晰的DSR数据防止过度借贷
3. **优化贷款方案**：了解不同贷款金额/期限的影响
4. **提升审批成功率**：提前了解DSR状态，优化申请策略

**间接价值（对公司）：**
1. **提高转化率**：用户使用计算器后，咨询意向提升60%+
2. **建立信任**：专业工具展示公司专业性
3. **收集线索**：可要求邮箱注册获取详细报告（可选）
4. **教育客户**：帮助客户理解DSR概念，降低咨询成本

**市场数据支持：**
- **90%+** 的贷款咨询网站包含DSR计算器
- **用户使用率**：40-60%的访客会使用计算器
- **转化率提升**：使用计算器的用户，咨询转化率提升 **2-3倍**

### 2. 房贷计算器 (Housing Loan / Mortgage Calculator)

#### 📌 **为什么最受欢迎？**

1. **高价值决策**：购房是人生最大投资之一，用户需要精确计算
2. **长期规划**：20-35年贷款期，需要清楚了解总成本
3. **对比需求**：用户需要对比不同银行/利率/期限
4. **压力测试**：用户想知道利率变化的影响

#### ⚙️ **详细功能规格**

**输入项（Input）：**
```
1. 房产信息 (Property Information)
   - 房产价格 (Property Price): RM X
   - 首付比例 (Down Payment %): X% 或 RM X
   - 贷款金额 (Loan Amount): 自动计算或手动输入

2. 贷款详情 (Loan Details)
   - 贷款利率 (Interest Rate): X% p.a.
   - 贷款期限 (Loan Tenure): X 年 (10-35年)
   - 利率类型 (Rate Type):
     * 固定利率 (Fixed Rate)
     * 浮动利率 (Variable Rate)
   - 还款方式 (Repayment Method):
     * 等额本息 (Equal Installment)
     * 等额本金 (Equal Principal)

3. 额外信息（可选）
   - 现有收入 (Monthly Income): RM X
   - 现有月供 (Existing Commitments): RM X
   - 计算DSR影响（可选）
```

**输出项（Output）：**
```
1. 还款计划 (Repayment Schedule)
   - 月供金额 (Monthly Installment): RM X
   - 总还款额 (Total Payment): RM X
   - 总利息 (Total Interest): RM X
   - 利息占本金比例 (Interest Ratio): X%

2. 逐年还款表 (Amortization Table)
   - 年份 (Year)
   - 累计本金 (Cumulative Principal)
   - 累计利息 (Cumulative Interest)
   - 剩余本金 (Remaining Principal)

3. 月供明细（可选展开）
   - 每月：本金、利息、剩余本金

4. 对比分析（如果输入收入）
   - DSR计算
   - 可负担性评估
   - 建议贷款金额

5. 利率变化影响（高级功能）
   - 利率+0.5%的影响
   - 利率+1%的影响
   - 利率+2%的影响
   - 可视化图表
```

**计算公式：**
```
1. 等额本息（最常见）：
   Monthly Rate = Annual Rate / 12
   Monthly Payment = Principal × [r(1+r)^n] / [(1+r)^n - 1]
   其中：r = 月利率, n = 总月数

2. 等额本金：
   Monthly Principal = Principal / n
   Monthly Payment = Monthly Principal + (Remaining Principal × Monthly Rate)
   （每月递减）

3. 总利息：
   Total Interest = (Monthly Payment × n) - Principal

4. 利息占比：
   Interest Ratio = (Total Interest / Principal) × 100
```

#### 💰 **客户获利点**

**直接价值：**
1. **精确规划**：清楚知道20-35年的还款计划
2. **成本对比**：了解不同利率/期限对总成本的影响
3. **首付优化**：计算最优首付比例
4. **提前还款分析**：了解提前还款节省的利息

**间接价值（对公司）：**
1. **高转化率**：房贷计算器用户，咨询转化率 **最高**（购房是重大决策）
2. **建立权威**：专业房贷计算器展示金融专业性
3. **后续服务机会**：房贷咨询、贷款申请、房产投资建议

**市场数据支持：**
- **95%+** 的房产/贷款网站包含房贷计算器
- **用户使用率**：50-70%的购房意向用户会使用
- **转化率**：使用房贷计算器的用户，咨询转化率 **3-5倍**高于普通访客

### 3. 信用卡还款模拟器 (Credit Card Payoff Simulator)

#### 📌 **为什么受欢迎？**

1. **高利息债务**：信用卡利率18-24%，用户急需解决方案
2. **还款策略对比**：用户想知道不同还款策略的差异
3. **时间成本**：清楚知道多久能还清，需要付多少利息
4. **债务整合评估**：对比最低还款 vs 整合贷款

#### ⚙️ **详细功能规格**

**输入项（Input）：**
```
1. 信用卡信息（多张）
   - 信用卡名称/银行 (Card Name/Bank)
   - 当前余额 (Current Balance): RM X
   - 年利率 (Annual Interest Rate): X% (默认18%)
   - 最低还款额 (Minimum Payment): RM X 或自动计算

2. 还款策略选择 (Repayment Strategy)
   - 策略1: 最低还款 (Minimum Payment Only)
   - 策略2: 积极还款 (Aggressive Repayment)
     * 每月还款额: RM X
   - 策略3: 债务整合贷款 (Consolidation Loan)
     * 整合贷款利率: X% p.a.
     * 整合贷款期限: X 年
```

**输出项（Output）：**
```
1. 策略对比表 (Strategy Comparison)
   - 策略1 (最低还款)
     * 还清时间: X 年 X 月
     * 总利息: RM X
     * 总还款额: RM X
     * 每月还款: RM X（最低）

   - 策略2 (积极还款)
     * 还清时间: X 年 X 月
     * 总利息: RM X
     * 总还款额: RM X
     * 每月还款: RM X（固定）
     * 节省时间: X 年 X 月
     * 节省利息: RM X

   - 策略3 (债务整合)
     * 还清时间: X 年
     * 总利息: RM X
     * 总还款额: RM X
     * 每月还款: RM X（固定）
     * 节省时间: X 年 X 月
     * 节省利息: RM X
     * 月供减少: RM X

2. 可视化对比
   - 三种策略的时间对比图
   - 三种策略的利息对比图
   - 每月还款额对比图

3. 建议与推荐
   - 最优策略推荐
   - 为什么推荐此策略
   - 下一步行动（如申请整合贷款）
```

**计算公式：**
```
1. 最低还款额（银行标准）：
   Minimum Payment = Max(Balance × 5%, RM 50)

2. 信用卡利息计算（复利）：
   Monthly Rate = Annual Rate / 12
   每月利息 = 当前余额 × Monthly Rate
   新余额 = 当前余额 + 利息 - 还款额

3. 还清时间计算（迭代）：
   While Balance > 0:
     利息 = Balance × Monthly Rate
     新余额 = Balance + 利息 - Payment
     月数++

4. 整合贷款月供（减余法）：
   Monthly Payment = Principal × [r(1+r)^n] / [(1+r)^n - 1]
```

#### 💰 **客户获利点**

**直接价值：**
1. **节省大量利息**：清晰看到整合贷款可节省的利息（通常节省50-70%）
2. **时间成本清晰**：了解不同策略的还清时间差异
3. **月供优化**：整合贷款通常月供减少30-50%
4. **决策依据**：数据驱动的还款策略选择

**间接价值（对公司）：**
1. **高转化率**：信用卡债务用户，咨询转化率 **非常高**（痛点强烈）
2. **服务机会**：信用卡管理服务、债务整合贷款申请
3. **建立信任**：帮助用户解决高利息债务问题

**市场数据支持：**
- **80%+** 的金融咨询网站包含信用卡计算器
- **用户使用率**：30-50%（信用卡债务用户）
- **转化率**：使用计算器后，咨询转化率提升 **2-4倍**
- **平均节省**：通过整合贷款，用户平均节省利息 **RM 20K-50K**

### 4. 贷款优化/整合计算器 (Loan Optimization / Consolidation Calculator)

#### 📌 **为什么受欢迎？**

1. **多重债务**：用户通常有多个贷款/信用卡
2. **优化需求**：希望降低月供、减少利息、改善DSR
3. **CTOS分析**：上传CTOS报告，获得专业分析
4. **对比分析**：清楚看到优化前后的差异

#### ⚙️ **详细功能规格**

**输入项（Input）：**
```
1. 收入信息
   - 月收入 (Monthly Income): RM X
   - 其他承诺 (Other Commitments): RM X

2. 现有债务列表（可添加多条）
   - 债务类型 (Debt Type):
     * 信用卡 (Credit Card)
     * 个人贷款 (Personal Loan)
     * 车贷 (Car Loan)
     * 房贷 (Housing Loan)
     * 其他 (Other)
   - 银行/机构 (Bank/Institution)
   - 未还余额 (Outstanding Balance): RM X
   - 利率 (Interest Rate): X% p.a.
   - 剩余期限 (Remaining Tenure): X 年
   - 月供 (Monthly Payment): RM X

3. 优化目标（可选）
   - 主要关注点 (Primary Concern):
     * 降低月供 (Reduce Monthly Payment)
     * 减少利息 (Reduce Interest)
     * 改善DSR (Improve DSR)
     * 快速还清 (Pay Off Faster)

4. CTOS上传（高级功能）
   - 上传CTOS报告PDF
   - 自动提取债务信息
```

**输出项（Output）：**
```
1. 当前财务状况 (Current Status)
   - 总债务余额: RM X
   - 总月供: RM X
   - 当前DSR: X%
   - 信用健康评分: X/100
   - 预计还清时间: X 年

2. 优化后财务状况 (Optimized Status)
   - 整合贷款金额: RM X
   - 新利率: X% p.a.
   - 新期限: X 年
   - 新月供: RM X
   - 新DSR: X%
   - 预计还清时间: X 年

3. 优化效果对比
   - 月供减少: RM X (减少 X%)
   - 利息节省: RM X (节省 X%)
   - DSR改善: -X%
   - 还清时间变化: ±X 年

4. 银行对比（可选）
   - 推荐银行列表
   - 各银行利率对比
   - 审批可能性评估

5. 详细建议
   - 哪些债务应该整合
   - 哪些债务应该保留
   - 优化策略说明
   - 下一步行动
```

**计算公式：**
```
1. 当前总月供：
   Total Monthly Payment = Σ(各债务月供)

2. 当前DSR：
   Current DSR = (Total Monthly Payment / Monthly Income) × 100

3. 整合贷款计算：
   Consolidation Amount = Σ(需整合的债务余额)
   新月供 = 使用减余法计算
   新DSR = (新月供 / Monthly Income) × 100

4. 节省计算：
   Monthly Savings = Current Monthly Payment - New Monthly Payment
   Total Interest Savings = Current Total Interest - New Total Interest
```

#### 💰 **客户获利点**

**直接价值：**
1. **大幅降低月供**：通常减少30-50%月供
2. **节省大量利息**：通过低利率整合，节省50-70%利息
3. **改善DSR**：降低DSR，提高新贷款审批可能性
4. **简化管理**：多个债务合并为一个，易于管理

**间接价值（对公司）：**
1. **高价值服务**：贷款整合是核心业务，转化率高
2. **专业形象**：CTOS分析展示专业能力
3. **长期客户**：整合贷款后，客户通常需要后续服务

**市场数据支持：**
- **70%+** 的贷款咨询网站包含优化计算器
- **用户使用率**：25-40%（多重债务用户）
- **转化率**：使用计算器后，咨询转化率提升 **3-5倍**
- **平均节省**：通过整合，用户平均月供减少 **RM 500-2000**

### 6. 投资回报计算器 (Investment Return Calculator)

#### 📌 **为什么受欢迎？**

1. **规划未来**：用户需要规划长期投资
2. **目标设定**：设定投资目标，计算所需投资
3. **对比分析**：对比不同投资方案
4. **复利效应**：展示复利的强大效果

#### ⚙️ **详细功能规格**

**输入项：**
```
- 初始投资: RM X
- 每月投资: RM X
- 投资期限: X 年
- 预期年回报率: X%
- 通胀率: X% (可选)
```

**输出项：**
```
- 未来价值: RM X
- 总投资: RM X
- 投资回报: RM X
- 回报率: X%
- 考虑通胀后的实际价值
- 逐年增长图表
```

#### 💰 **客户获利点**

- **目标规划**：清楚知道需要投资多少才能达到目标
- **复利理解**：理解长期投资的价值
- **方案对比**：对比不同投资策略
