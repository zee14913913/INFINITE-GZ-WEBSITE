## 📝 步骤（只需要3步）

### 步骤 1：找到您的视频文件

找到您想要使用的视频文件，例如：
- 文件名可能是：`my-video.mp4` 或 `video.mov` 等等
- 视频文件可以在您的电脑的任何位置

### 步骤 2：重命名视频文件

将视频文件重命名为：**`creditpilot-hero-bg.mp4`**

**如何重命名（Mac）：**
1. 找到视频文件
2. 点击文件名称（选中文件）
3. 按 `Enter` 键或点击文件名
4. 删除旧名称，输入：`creditpilot-hero-bg.mp4`
5. 按 `Enter` 确认

**如何重命名（Windows）：**
1. 右键点击视频文件
2. 选择"重命名"
3. 输入：`creditpilot-hero-bg.mp4`
4. 按 `Enter` 确认

### 步骤 3：复制视频文件到项目文件夹

**方法A：使用 Finder（Mac）或文件管理器（Windows）**

1. 打开 Finder（Mac）或文件管理器（Windows）

2. 找到项目文件夹：
   ```

   ```

3. 进入以下文件夹：
   ```
   infinite-gz-temp/
     └── public/
         └── videos/
   ```

4. 将重命名后的视频文件 `creditpilot-hero-bg.mp4` 复制到这个 `videos` 文件夹里

**最终文件路径应该是：**
```
/Users/1491-3913zee/Desktop/infinite-gz-temp/public/videos/creditpilot-hero-bg.mp4
```

**方法B：使用终端命令（如果您熟悉终端）**

打开终端，运行以下命令：

```bash

# 注意：将 /path/to/your/video.mp4 替换为您视频文件的实际路径
```

## ❓ 常见问题

### Q1: 视频格式不是 MP4 怎么办？

**A:** 需要转换为 MP4 格式

**使用在线转换工具：**
1. 访问：https://cloudconvert.com/
2. 上传您的视频文件
3. 选择输出格式为 MP4
4. 下载转换后的文件

**使用 FFmpeg（命令行）：**
```bash
ffmpeg -i input.mov -vcodec h264 -acodec aac output.mp4
```

### Q2: 视频文件太大怎么办？

**A:** 需要压缩视频文件

**使用在线压缩工具：**
1. 访问：https://www.freeconvert.com/video-compressor
2. 上传视频文件
3. 压缩后下载

**使用 FFmpeg（命令行）：**
```bash
ffmpeg -i input.mp4 -vcodec h264 -acodec aac -b:v 2M output.mp4
```

### Q3: 视频不显示怎么办？

**检查清单：**
- ✅ 文件名是否正确：`creditpilot-hero-bg.mp4`（完全匹配，包括大小写）
- ✅ 文件是否在正确位置：`public/videos/creditpilot-hero-bg.mp4`
- ✅ 文件格式是否是 MP4
- ✅ 浏览器控制台是否有错误信息（按 F12 打开开发者工具查看）

### Q4: 如何修改视频的透明度或效果？

视频效果的代码在：`app/creditpilot/page.tsx`

**调整视频透明度（第28行）：**
```tsx
// 当前是 50% 透明度

// 改为 30% 透明度（更暗）
className="w-full h-full object-cover opacity-30"

// 改为 70% 透明度（更亮）
className="w-full h-full object-cover opacity-70"
```

**调整渐变遮罩（第36行）：**
```tsx

// 更暗的遮罩
```
